SLM - Tool Documentation

Mapping Tools

Canopy Cost Raster

Creates a canopy raster and a cost raster from a CHM input raster. The output rasters are used for subsequent SLM tools.

Parameter Description Type Default
Canopy Height Model (CHM) Raster Input CHM raster file used as basis for mapping seismic lines and their footprint. This layer should be projected in any Universal Transverse Mercator (UTM) coordinate system. The data should be preferably derived from high resolution LiDAR or photogrammetry (>25 pts/mÂ²). TIF
Canopy Height Threshold Height threshold in meters above which CHM pixels are considered canopy. number 1
Tree Search Radius Radius of canopy influence in the cost raster. This factor smooths the canopy in the cost raster which helps to prevent gaps in sparsely vegetated areas (e.g. wetlands) to be incorrectly identified as footprint. A large search radius (>=5m) may cause excessive smoothing of the cost raster which can lead to least cost paths ignoring seismic-line nuances in sparsely vegetated forests. A small radius (<=1m) may cause the least cost path to cut corners through small gaps in sparse vegetation as well as avoid small obstacles that should not affect overall line shape. number 3
Maximum Line Distance Maximum euclidean distance from canopy. This is a second smoothing factor which helps to position the seismic line shape in the center of the line footprint. An excessively small (<=1m) or large (>20m) value may cause the center line to be positioned close to one of the edges of the footprint. number 10
Canopy Avoidance Ratio of importance between canopy search radius and euclidean distance. A value close to zero (0) prioritizes search radius whereas a value close to one (1) prioritizes euclidean distance. A small value (<=0.1) may cause the seismic lines to miss nuances in sparsely vegetated terrain whereas a big value (>=0.5) may overemphasize turns in complex trails. This factor influences final footprint size in a minor way. number 0.3
Cost Raster Exponent Affects the cost of vegetated areas in an exponential fashion. A low (<=1) exponent may lead to lines cutting through corners, whereas a large (>=3) exponent may lead to least cost paths completely avoiding narrow lines. number 1.5
Output Canopy Raster Output raster classified as canopy (1) and non-canopy (0). TIF
Output Cost Raster Output cost raster used in subsequent SLM tools for least-cost analysis. TIF
Center Line

Determines the least cost path between vertices of the input lines.

Parameter Description Type Default
Seismic Lines Feature Class Input polyline shapefile. SHP
Cost Raster Input raster image used to calculate the least cost path. TIF
Line Processing Radius Maximum processing distance from input lines. A large search radius may increase processing times whereas a small radius may cause undesired clipping. number 35
Output Center-Line Output center-line shapefile. SHP
Line Footprint

Creates footprint polygons for each input line based on a least cost corridor method and individual line thresholds.

Parameter Description Type Default
Center-lines Feature Class Input polyline shapefile. SHP
Canopy Raster Input raster image used to exclude canopy from line footprint. TIF
Cost Raster Input raster image used to calculate the least cost corridor. TIF
Corridor Threshold Field Name of the field (within the input lines) that indicates the least cost corridor thresholds (LCCT). text CorridorTh
Maximum Line Width Maximum processing width for input lines. A large value may increase processing times whereas a small value may cause undesired clipping. number 32
Expand And Shrink Cell Range Range used for cell erosion before final polygons are generated. Useful to remove small artifacts. number 3
Process Segments If set to False, will process each line from start to end ignoring midpoints. If set to True, will process each segment between each vertex of the input lines separately. The default is False, since it is assumed that the input lines for this tool are manually corrected center-lines. If using regional-scale (1:20,000) lines as input this may be set to True. True/False False
Output Shapefile Output footprint polygons. SHP

Attribute Tools

Seismic Line Attributes

Calculates a series of attributes related to seismic line shape, size and microtopography.

Parameter Description Type Default
Input Lines Input seismic line center shapefile which will be segmented and attributed. SHP
Input Footprint Polygons Input seismic line footprint shapefile used to derive spatial attributes. SHP
Input Raster CHM Input canopy height model raster used to derive microtopography attributes. TIF
Sampling Type How the input lines are segmented for attribution: WHOLE-LINE, the entire extent of each line is attributed without subdivisions; LINE-CROSSINGS, each line is split at line intersections; ARBITRARY, each line is segmented using an arbitrary length. text LINE-CROSSINGS
Segment Length Arbitrary segment length (m). If the Sampling Type field is not set as ARBITRARY this field is ignored. number 50
Line Split Tolerance Tolerance radius (m) used to split lines. If the Sampling Type field is set as WHOLE-LINE this field is ignored. number 3
Maximum Line Width Maximum line width (m) used to search for surrounding footprint. number 25
Output Attributed Segments Output features that will be created. SHP
Raster Line Attributes

Samples a raster image along lines and assigns cell statistics to each line.

Parameter Description Type Default
Input Lines Input seismic line center shapefile which will be segmented and attributed. SHP
Input Raster Variable Input raster image that will be sampled to attribute the input lines. TIF
Sampling Type How the input lines are segmented for attribution: WHOLE-LINE, the entire extent of each line is attributed without subdivisions; LINE-CROSSINGS, each line is split at line intersections; ARBITRARY, each line is segmented using an arbitrary length. text LINE-CROSSINGS
Sampling Interval Sampling interval along input lines in meters. number 5
Segment Length Arbitrary segment length (m). If the Sampling Type field is not set as ARBITRARY this field is ignored. number 50
Line Split Tolerance Tolerance radius (m) used to split lines. If the Sampling Type field is set as WHOLE-LINE this field is ignored. number 3
Sampling Method Method used to handle samples when attributing lines: Minimum, Maximum, Mean, Standard Deviation, Median, Mode, or Range. text Mean
Output Attributed Segments Output features that will be created. SHP

Auxiliary Tools

Clip Shape By Raster

Extracts input features that overlay raster cells which are not NoData. This is useful to prepare input seismic lines for corridor analysis, since vertices must overlap with valid cells for least cost methods.

Parameter Description Type Default
Shapefile Input features to be clipped. SHP
Clip Raster Input raster used to clip the input features. TIF
Shrink Size Number of cells to shrink in order to avoid raster edge contact. number 1
Output Shapefile Output features that will be created. SHP
Split By Polygon

Splits the input lines where they intersect the edges of the input polygons.

Parameter Description Type Default
Input Lines Input polyline shapefile to be split. SHP
Clip Polygons Input polygon shapefile that is used to split the input lines. TIF
Output Lines Output features that will be created. SHP
Zonal Threshold

Assigns corridor thresholds to the input lines based on their surrounding canopy density.

Parameter Description Type Default
Input Lines Input polyline shapefile for which each feature will be assigned an unique corridor threshold. SHP
Corridor Threshold Field Name of the field (within the input lines) that indicates the corridor thresholds. text CorridorTh
Canopy Raster Input canopy raster used to estimate canopy density. TIF
Canopy Search Radius Search radius around line features where zonal statistics are calculated. number 20
Minimum Value Minimum value for corridor threshold. number 4
Maximum Value Maximum value for corridor threshold. number 20
Output Lines Output features that will be created. SHP

Cost Raster Input raster image used to calculate the cost corridor. TIF
